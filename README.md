# Cart
## Overview
基于OpenCart进行二次开发的商城框架，重新设计了整个项目的目录结构，使整个项目更清晰，操作更简单。封装了几个功能类和一些
多次复用的代码，使得增加新页面更简单，结构更明了。
## How to use
基本用法与OpenCart基本相同，请参照[How to documents](http://docs.opencart.com/)
